Logistic Regression from Scratch (NumPy)

This project implements logistic regression from scratch using NumPy, without using machine learning libraries such as Scikit-learn.

Developed as part of CAP 4630 (Machine Learning).

Features
	•	Sigmoid function implementation
	•	Cross-entropy cost function
	•	Gradient descent optimization
	•	Decision boundary visualization
	•	L1 and L2 regularization

Results
	•	Achieved ~85% training accuracy on classification dataset

Technologies
	•	Python (NumPy, Pandas, Matplotlib)
